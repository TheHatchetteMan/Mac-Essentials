# Mac-Essentials

A personal repository to help you quickly set up a new Mac with your favorite applications and essential tools.

## Whatâ€™s in this repository?

- **apps.md:** List of useful apps you like to install on every new Mac.
- **setup/brew.sh:** Script for automating installation using Homebrew.
- **setup/post-setup.md:** Manual notes, settings, or post-install steps.

## How to use

1. Review and update `apps.md` and `brew.sh` to match your preferences.
2. Run the Brew script on your new Mac:

    ```sh
    bash setup/brew.sh
    ```
3. Follow any extra instructions in `setup/post-setup.md`.
